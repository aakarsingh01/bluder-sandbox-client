CodeSandbox - Running in Offline Mode

üö´ External API Blocked

This CodeSandbox instance is configured to run completely offline. External CodeSandbox APIs have been blocked for security and privacy.

You are accessing a locally hosted version of CodeSandbox that:

  • Works entirely offline without internet connectivity
  • Blocks all external CodeSandbox API calls
  • Allows NPM package installation from registry
  • Provides a secure, isolated development environment
  • Runs completely on your local infrastructure

To use this offline CodeSandbox:

# Run with Docker Compose docker-compose up # Or build and run manually docker build -t codesandbox-offline . docker run -p 3000:80 codesandbox-offline

Note: Some features requiring external CodeSandbox services (authentication, cloud sync, templates) are intentionally disabled in this offline version.
